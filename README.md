terminal-find
=============

faster way to search the web

# usage

copy web-find.py to /bin/ :

```terminal
$ cp web-find.py /bin/
```

rename to web.find : 

```terminal
$ mv /bin/web-find.py /bin/web-find
```

now you can use this script in interactive mode:

```terminal
$ web-find
web-finder > [provider] [search]
```

# example

```terminal
web-find > google github projects
```

# Help

```terminal
web-find > help
```
